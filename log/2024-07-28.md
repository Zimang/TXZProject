## 工时管理系统

### 数据

1. **proj**
   1. name 
   2. start time
   3. end time
   4. duel time
   5. work time
   6. info
   7. tags 标签用于过滤
   8. resume
   9. **phases**
      1. name
      2. start time
      3. end time 
      4. duel time
      5. work time
      6. doc
      7. info
      8. **tasks**
         1. name 
         2. s_t 开始时间
         3. e_t 结束时间
         4. d_t 计划结束时间
         5. w_t 总工时
         6. **time pair** 时间片
            1. s_t 开始时间 精确到分
            2. e_t 结束时间 精确到分
            3. interval 精确到分
